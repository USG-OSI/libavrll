# libavrll

libavrll is a bare-metal AVR low-level driver library built for register-aware embedded development.

The project is designed to provide reusable peripheral interfaces for GPIO, external interrupts, delays, timers, PWM, UART, SPI, I2C, ADC, watchdog, and EEPROM, while keeping the implementation close to the hardware.

This repository is currently in early development. Initial structure, module organization, and documentation are being set up.
